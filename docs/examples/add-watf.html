WebAssembly text format - demo

instantiateFromSource is not part of WebAssembly's API, it's only meant for testing purpose.

Input

WATF 
          
(module
 (func $add (param i32) (param i32) (result i32)
  get_local 0
  get_local 1
  i32.add)
 (export "add" (func $add))
)

Client

JavaScript 
          
var code = document.getElementById("input").innerText;
var module = webassemblyInterpreter.instantiateFromSource(code);

var res = module.exports.add(1, 1);
document.getElementById("res").innerHTML = res;
